研究会の活動を通じて触れた技術

菊地伸一

私は社会人で現職のWebアプリケーションエンジニアです。 普段の業務ではPHPをメインで扱っており、自分の担当領域外の技術に触れる機会はなかなかありません。

東大エンジニアリング研究会の活動では、普段は全く触れない領域の技術について知る機会がたくさんあります！

本記事では、自分が触った中で特に印象的な技術を2つ紹介したいと思います。

3Dモデル

まずは3Dモデルについての紹介です。 3Dモデリングでは有名なツールである「Blender」を用いて、仮面ライダーのマスク造形に挑んでみました！

マスク制作では型として造形したい仮面ライダーのマスク画像を配置し、画像に沿って3D立体を配置・変形させています。

ゲームや映像作品で目にする3Dモデルはこのように作られていたのか！と感心するとともに、その難易度を身をもって知ることが出来ました。

Flutterを用いたネイティブアプリ開発

続いて、Flutterを用いたネイティブアプリ開発についての紹介です。

クロス開発プラットフォームであるFlutterでは、通常は開発言語が異なるAndroidとiOSを一つの言語で開発できます。 ちょうど私がAndroidとiPhoneの二台持ちという事情もありFlutterに挑んでみました！

画面の状態管理やウィジェット構成といった、普段の仕事では扱わない技術や概念に触れて知見を広げることが出来ました。
