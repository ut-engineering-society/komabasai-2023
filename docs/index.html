東大エンジニアリング研究会

「実行のためのコミュニティ」を合言葉に掲げる、開発・研究・学習の相互補助を目指す開発コミュニティです。

少人数のセミナーによる進捗報告会や特定のテーマに関する勉強会などを通し、エンジニアリングに関する知見を共有し、お互いに高め合えるよう活動しています。
